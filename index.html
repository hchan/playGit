To create a new EGit Project

1) New->Other->Git-Git Repository

Choose your parent i.e. c:\workspace
Choose your projectName i.e. playGit

2) Login to GitHub
create a playGit project

3) Create a New java project
c:\workspace\playGit

4) Team->Commit (fill in information from 2)



-----------------------
or
1) Create a Java project first
c:\workspace\playGit

2) from the commandline
cd \workspace\playGit
git init (this is assuming you have the Git binary installed http://git-scm.com/download/win)

3) Login to GitHub
create a playGit project

4) Team->Commit (fill in information from 3)
